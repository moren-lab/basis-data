Apa itu SQL?

SQL (Structured Query Language) adalah bahasa standar yang digunakan untuk mengelola dan memanipulasi data dalam sistem basis data relasional atau RDBMS (Relational Database Management System). Dengan SQL, kita dapat membuat, menghapus, mengubah, serta mengambil data dari tabel dalam sebuah database. SQL memiliki peran penting karena menjadi jembatan komunikasi antara pengguna dan sistem basis data, sehingga seluruh proses pengolahan data dapat dilakukan dengan efisien dan terstruktur. Secara umum, SQL dibagi menjadi beberapa jenis perintah utama, yaitu DDL (Data Definition Language), DML (Data Manipulation Language), DQL (Data Query Language), DCL (Data Control Language), dan TCL (Transaction Control Language). DDL digunakan untuk mendefinisikan struktur database, misalnya membuat tabel menggunakan perintah CREATE TABLE, menambah kolom dengan ALTER TABLE, atau menghapus tabel dengan DROP TABLE. DML berfungsi untuk memanipulasi data di dalam tabel, misalnya menambah data dengan INSERT, mengubah data dengan UPDATE, dan menghapus data dengan DELETE. DQL digunakan untuk mengambil atau menampilkan data dari tabel menggunakan perintah SELECT, misalnya SELECT * FROM mahasiswa untuk menampilkan seluruh data mahasiswa.
